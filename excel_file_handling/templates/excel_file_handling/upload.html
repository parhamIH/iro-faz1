نکات مهم
  • title: عنوان محصول (اجباری)
  • brand: نام برند (باید قبلاً ثبت شده باشد)
  • categories: نام دسته‌بندی (باید قبلاً ثبت شده باشد)
  • tags: تگ‌ها با کاما جدا می‌شوند
ترتیب import: ابتدا برندها و دسته‌بندی‌ها را import کنید
مثال: brand="سامسونگ" (نام دقیق برند)
دانلود قالب
ترتیب صحیح import:
  1. برندها → ابتدا برندها را import کنید
  2. دسته‌بندی‌ها → سپس دسته‌بندی‌ها را import کنید
  3. رنگ‌ها و گارانتی‌ها → سپس رنگ‌ها و گارانتی‌ها را import کنید
  4. محصولات → در نهایت محصولات را import کنید
مثال کامل و مرتبط

در اینجا یک مثال کامل از داده‌های مرتبط نشان داده می‌شود:

1. برندها
namedescription
سامسونگشرکت سامسونگ
اپلشرکت اپل
2. دسته‌بندی‌ها
nameparent
الکترونیکخالی
موبایلالکترونیک
3. محصولات
titlebrandcategories
Galaxy S21سامسونگموبایل
iPhone 15اپلموبایل
نحوه ارتباط داده‌ها:
  • برند → محصول: در ستون "brand" محصول، نام دقیق برند را بنویسید
  • دسته‌بندی → محصول: در ستون "categories" محصول، نام دقیق دسته‌بندی را بنویسید
  • دسته‌بندی والد → فرزند: در ستون "parent" زیردسته، نام دسته‌بندی والد را بنویسید
  • تگ‌ها: با کاما جدا کنید: "گوشی,هوشمند,سامسونگ"
خطاهای رایج:
  • "نام برند یافت نشد": ابتدا برند را import کنید
  • "نام دسته‌بندی یافت نشد": ابتدا دسته‌بندی را import کنید
  • "دسته‌بندی والد یافت نشد": ابتدا دسته‌بندی والد را import کنید
  • نام‌ها باید دقیقاً مطابقت داشته باشند (حساس به حروف بزرگ و کوچک)
